# 🎮 Roblox Module Uploader
Uploads modules through Roblox's API, store the requires on a website (optional)

## ⚙️ Running

```sh
# Install dependencies
$ pip install -r requirements.txt

# Run main script
$ python3 main.py

# Run website script (optional)
$ python3 web/main.py
```
